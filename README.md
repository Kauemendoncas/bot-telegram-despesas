# 🤖 Bot Telegram - Analisador de Gastos Pessoais

<p align="center">
  Um bot desenvolvido em Python para auxiliar no controle e organização de despesas pessoais através do Telegram.
</p>

## 📌 Sobre o projeto

O **Bot Telegram - Analisador de Gastos** é uma aplicação desenvolvida em Python com o objetivo de facilitar o registro e gerenciamento de despesas do dia a dia.

Através de uma conversa simples pelo Telegram, o usuário consegue informar seus gastos, registrar descrições e organizar suas informações financeiras de maneira prática.

O projeto foi desenvolvido aplicando conceitos de **Programação Orientada a Objetos (POO)**, separação de responsabilidades e integração com uma API externa.

---

## 🎯 Objetivo

O principal objetivo deste projeto é criar uma ferramenta simples e acessível para ajudar usuários a terem maior controle sobre seus gastos pessoais.

# ⚙️ Como executar o projeto

## 1. Clone o repositório

```bash
git clone https://github.com/Kauemendoncas/bot-telegram-despesas.git
```

## 2. Entre na pasta do projeto

```bash
cd bot-telegram-despesas
```

## 3. Instale as dependências

```bash
pip install -r requirements.txt
```

## 4. Configure o token do Telegram

Crie um bot através do **BotFather** no Telegram e adicione o token no arquivo de configuração.

⚠️ Nunca compartilhe seu token publicamente.

## 5. Execute o projeto

```bash
python bot.py
```
